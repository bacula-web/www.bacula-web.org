I'm pleased to announce that Bacula-Web 9.9.0 release is now available

<!-- truncate -->

## Release Highlights

- Changed speed column to bit rate in Job report page
- improved the way we validate user input (especially from form)
- ensure username does not already exist while creating user
- fixed volumes total bytes in Volumes report page
- many bug fixes

See [v9.8.0 to v9.9.0](https://github.com/bacula-web/bacula-web/compare/v9.8.0...v9.9.0) for all changes details.

## Release notes

Check the [release notes](https://github.com/bacula-web/bacula-web/releases/tag/v9.9.0) on GitHub.

:::tip
As usual, I strongly encourage everyone to upgrade to the latest release.
